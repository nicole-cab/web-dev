Numbers

numbers as values can represent different things, for example for the property line-height its unitless value would represent the ratio proportional to the element's pixel font size, e.g. if font-size: 24px then line-height: 1.5 would be 150% of 24px which is 36px

filter: sepia(0.5) applies a 50% sepia filter

opacity: 0.5 applies a 50% opacity

rgb(50, 50, 50) represent the rgb values

transform: scale(1.2) scales the element by 120% of its intial size

Percentages

percentages in width, margin or padding is the percentage of the parent's width

percetages in transform is based on the element applying a transformation, e.g. if its width is 150px than transform: translateX(10%) would be 15px
